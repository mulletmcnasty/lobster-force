## How to Deploy Landing Page

### Option 1: GitHub Pages
1. Create repo: `lobsterforce/lobsterforce.github.io`
2. Push `index.html` to main branch
3. Enable GitHub Pages in repo settings
4. Site live at `lobsterforce.github.io`

### Option 2: Custom Domain
1. Buy `lobsterforce.com`
2. Point DNS to GitHub Pages or hosting provider
3. Deploy `index.html`
4. Add SSL certificate

### Option 3: Netlify Drop
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag `index.html` into browser
3. Instant live site with custom domain support
